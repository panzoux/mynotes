MyNotes — スマホから疑問を即ノート化するシステム

2026-06-13
参考: スマホから疑問を投げたらHTMLノートが増えていく仕組みを作った — Zenn

技術的な疑問を、その場でAIに投げてHTMLページとして蓄積。「スマホで調べる」から「スマホで資産化する」への体験シフトが目的。

なぜ作るのか

スマホで検索して調べた内容は「その場でわかった気になる」が、記憶に残りにくい。書き起こす手間を省く。

システム構成

スマホからの接続方法（remote-control）

Claude Code の remote-control 機能でスマホのClaudeアプリからPCを直接操作できる。git操作を含めすべてスマホから完結する。

PC側（操作前に起動しておく）

cd ~\source\repos\panzoux\mynotes\
claude /login
claude> remote-control

スマホ側

  1. Claudeアプリを開く
  2. 「リモートコントロールセッションに接続」を選択
  3. PCに表示されたコードを入力して接続
  4. あとは普通にチャット — PCのファイル操作・git操作がスマホから動く

ショートコマンド

入力 動作
追加: 〇〇とは ノート新規作成 → commit
追加 #タグ: 〇〇とは タグ指定付きで追加
一覧 / list ノート一覧を表示
編集: ノート名 / edit: name 指定ノートを編集

技術スタック

  • フロントエンド: HTML / CSS / SVG のみ（ビルド不要）
  • AI: Claude Code（remote-control経由）
  • ホスティング: GitHub Pages（無料）
  • バージョン管理: Git / GitHub

ディレクトリ構成

mynotes/
├── index.html        ← 一覧ページ（タグフィルター付き）
├── style.css
├── CLAUDE.md         ← AIへの指示・ルール定義
├── .gitignore        ← 秘匿ファイル除外
└── notes/
    └── *.html        ← 各ノート

ガードレール

  • git push は必ず人間が差分を目視確認してから実行（AIは commit 止まり）
  • CLAUDE.md と .gitignore で秘匿情報を管理外に
  • JSON / YAML / ENV ファイルはコミット対象外
  • 参照元 URL を各ノートに必ず記載
