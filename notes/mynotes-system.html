MyNotes — スマホから疑問を即ノート化するシステム

2026-06-12
参考: スマホから疑問を投げたらHTMLノートが増えていく仕組みを作った — Zenn

移動中や就寝前に浮かんだ技術的な疑問を、その場でAIに投げてHTMLページとして蓄積していく仕組み。「スマホで調べる」から「スマホで資産化する」への体験シフトが目的。

なぜ作るのか

スマホで検索して調べた内容は「その場でわかった気になる」が、後から記憶に残りにくい。ノートとして書き起こす手間が大きいため、日常の疑問が資産にならずに流れていく。このシステムはその摩擦をゼロにする。

システム構成

技術スタック

  • フロントエンド: HTML / CSS / SVG のみ（ビルド不要）
  • AI: Claude（このサイトの場合）/ 元記事は OpenAI Codex
  • ホスティング: GitHub Pages（無料・静的ファイルをそのまま配信）
  • バージョン管理: Git / GitHub

ディレクトリ構成

mynotes/
├── index.html        ← 一覧ページ（タグフィルター付き）
├── style.css
├── CLAUDE.md         ← AIへの指示・ルール定義
├── .gitignore        ← 秘匿ファイル除外
└── notes/
    └── *.html        ← 各ノート

ガードレール

  • git push は必ず人間が差分を目視確認してから実行（AIは commit 止まり）
  • CLAUDE.md と .gitignore で秘匿情報を管理外に
  • JSON / YAML / ENV ファイルはコミット対象外
  • 参照元 URL を各ノートに必ず記載

使い方（日常のフロー）

  1. スマホの claude.ai でプロジェクトを開く
  2. 追加: 〇〇とは と送信
  3. Claude がノートを生成・commit
  4. PC で差分を確認して git push
  5. GitHub Pages に自動反映
