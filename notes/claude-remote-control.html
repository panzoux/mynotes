Claude remote-control でスマホからPCを操作する

2026-06-13
参考: 自分で発見

Claude Code の remote-control 機能を使うと、スマホのClaudeアプリからPCのClaude Codeセッションに接続して操作できる。git commit / push などローカル操作がスマホから完結する。

手順

PC側（1回やっておけばOK）

cd ~\source\repos\panzoux\mynotes\
claude /login
claude> remote-control

待機状態になり、接続コードが表示される。

スマホ側

  1. Claudeアプリを開く
  2. 「リモートコントロールセッションに接続」を選択
  3. PCに表示されたコードを入力して接続
  4. あとは普通にチャットするだけ — PCのファイル操作・git操作がスマホから動く
